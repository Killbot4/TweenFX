## What is TweenFX?  

TweenFX is a straightforward animation tool built for Godot, a popular game engine. It helps you create smooth movements and effects without needing to write complex code. If you want your game or app to have neat animations that catch the eye, TweenFX makes the process simple and clear.

## 💻 System Requirements  

- Windows 10 or later  
- Godot Engine version 4.0 or higher installed  
- Internet connection to download files  
- At least 2 GB of free disk space

## 🌟 Features  

- Easy setup for tween animations in Godot  
- Supports moves, fades, scale changes, and more  
- Works smoothly with Godot’s built-in tools

## 🔧 Using TweenFX in Godot  

TweenFX helps you add motion and effects to objects. Here are some simple ways you can use it:  

- Move objects smoothly from one spot to another  
- Make sprites fade in or out  
- Scale objects bigger or smaller for emphasis  
- Combine these effects for more visual appeal

Just select the object you want to animate in Godot and choose the TweenFX options available. Follow simple steps in Godot’s editor to set the tween’s start and end values.
